Principle

Your personal data will be treated confidentially. All actions are bound to the legal principles of the Hessian Data Protection Act (HDSG), the Telemedia Act (TMG), the EU Data Protection Basic Regulation (EU-DSGVO) and this data protection declaration. This is guaranteed by TransMIT GmH as operator of »frag.jetzt«.

Usage data

We collect, process and use personal data about the use of »frag.jetzt« (usage data) only to the extent necessary to enable the user to use »frag.jetzt«. Normally a registration for the use of »frag.jetzt« is not necessary.

There are two exceptions: first, the instructor wants to save his sessions on the server of »frag.jetzt« permanently; and second, a user should be included as a moderator in a session. In both cases, a valid e-mail address must be used as the login name. A lecturer, who includes a moderator in his session, has knowledge of the moderator's mail address.

If an account is not used within 180 days, the account will be automatically deleted without notice.

Access data

Personal data will only be collected to the technically necessary extent of »frag.jetzt«. Apart from the e-mail address as the login name of a registered user, no personal data is stored.

Log files

We automatically collect and store information in log files that your browser automatically transmits to us. These are: browser type and browser version, operating system used, referrer URL, hostname of the accessing computer and time of the server request. This data cannot be assigned to specific persons. This data is not merged with other data sources.

Matomo

This offer uses Matomo (formerly Piwik), an open source software for statistical analysis of the accesses, see matomo.org Opens new window. Matomo uses cookies, which are stored on your computer and which allow an analysis of the usage of »frag. jetzt«. The information generated by the cookies about the use of this offer is stored on the server of the provider in Germany. The IP address is anonymized immediately after processing and before saving it by shortening the IP address. A conclusion on your person is not possible for us. You can find further information on the Matomo terms of use and the data protection regulations at https://matomo.org/privacy/ Opens new window.

Web Storage

»frag.jetzt« uses the so-called web storage technology. This means that data is stored locally in the cache of your browser and can continue to exist and be read even after you close the browser window or quit the program - unless you delete the cache.

DeepL.com

The concept of »frag.jetzt« involves the back translation of a question by the online translation service Deepl.com, see our explanation in the introduction. If you click on the »Check text« button when entering a question, the text will be sent to the API of DeepL.com. This is done via an encrypted connection. About the compliance with data protection by DeepL.com you can inform yourself on the following page: https://www.deepl.com/de/privacyopens new window.

Data security

Comprehensive technical and operational protective measures have been taken to protect your data from accidental or intentional manipulation, loss, destruction or access by unauthorized persons. The security procedures are regularly reviewed and adapted to technological progress. However, data transmissions on the Internet (e.g. when communicating by e-mail) can have security gaps, so that complete protection of the data against access by third parties is not possible.

Insofar as external service providers come into contact with your personal data, legal, technical and organisational measures as well as regular checks will ensure that this data is not passed on to third parties. comply with the provisions of the data protection laws.
