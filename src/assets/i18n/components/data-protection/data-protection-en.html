Principle

Your personal data will be treated confidentially. All actions are bound to the legal principles of the Hessian Data Protection Act (HDSG), the Telemedia Act (TMG), the EU Data Protection Basic Regulation (EU-GDPR) and this data protection declaration. This is guaranteed by TransMIT GmbH as operator of »frag.jetzt«.

Usage data

We collect, process and use personal data about the use of »frag.jetzt« (usage data) only to the extent necessary to enable the user to use »frag.jetzt«. Normally a registration for the use of »frag.jetzt« is not necessary.

There are two exceptions: firstly, the room creator wants to permanently store his/her own rooms on the server of »frag.jetzt«, and secondly, a user is to be permanently accepted as a moderator of a room. In both cases, a valid e-mail address must be used as the login name. A lecturer, who includes a moderator in his/her room, has knowledge of the moderator's mail address.

Rooms and bonuses created as a guest, i.e. anonymously without registration, are deleted as soon as you delete the web storage of your browser. Rooms created by registered users are part of their accounts and can be deleted by them. Rooms created with an account will be deleted 180 days after the last visit / login.

Access data

Personal data will only be collected to the technically necessary extent of »frag.jetzt«. Apart from the e-mail address as the login name of a registered user, no personal data is stored.

External hosting

»frag.jetzt« is hosted by the external service provider Hetzner Online GmbHOpens a new tab on a server in Germany. The personal data collected on this website is stored on the hoster's servers. The use of a professional hoster is in the interest of a secure, fast and reliable provision of our online offer (Art. 6 para. 1 lit. f GDPR).

Hetzner Online GmbH has an annual audit report prepared by TÜV Rheinland on its behalf. This ensures that all security standards according to Annex 2 of the AV contract (technical and organizational measures according to Art. 32 GDPR and Annex) are met.

Log files

We automatically collect and store information in log files that your browser automatically transmits to us. These are: browser type and browser version, operating system used, referrer URL, hostname of the accessing computer and time of the server request. This data cannot be assigned to specific persons. This data is not merged with other data sources. We record the log data of the »frag.jetzt« web server for a maximum of 30 days.

Matomo

This offer uses Matomo (formerly Piwik), an open source software for statistical analysis of the accesses. Matomo uses cookies, which are stored on your computer and which allow an analysis of the usage of »frag.jetzt«. The information generated by the cookies about the use of this offer is stored on the server of »frag.jetzt« in Germany. The IP address is anonymized immediately after processing and before saving it by shortening the IP address. A conclusion on your person is not possible for us. You can find further information on the Matomo terms of use and the data protection regulations at https://matomo.org/privacy/ Opens a new tab.

Matomo cookies remain on your end device until you delete them. The storage of Matomo cookies and the use of this analysis tool are based on Art. 6 para. 1 lit. f GDPR. The website operator has a legitimate interest in the anonymized analysis of user behavior in order to optimize its website. The information generated by the cookie about the use of this website will not be disclosed to third parties.

Web Storage

»frag.jetzt« uses the so-called web storage technology. In this process, data is stored locally in your browser's cache for an unlimited period of time, and can still exist and be read after you close the browser window or exit the program - as long as you do not delete the cache.

In your browser's web storage, we store the following usage data: Acceptance of our privacy policy, read confirmation of the "frag.jetzt" news, selected language setting (German, English or French), completion of the onboarding tour, display settings, whether logged in as guest or registered user, user ID and, if logged in, the mail address as login name, visited rooms and taken role (room creator, moderator or participant), and whether a post has been written (only those who have written at least one post can rate the »frag.jetzt« web app). This data can be viewed via the browser tools and deleted locally.

DeepL.com

The concept of »frag.jetzt« involves the back translation of a question by the online translation service Deepl.com, see our explanation in the introduction. If you click on the »Optimize text« button when entering a question, only the text will be sent to the API of DeepL.com. Your IP address will not be transmitted. The API request is made over an encrypted connection. About the compliance with data protection by DeepL.com you can inform yourself on the following page: https://www.deepl.com/de/privacyOpens a new tab.
