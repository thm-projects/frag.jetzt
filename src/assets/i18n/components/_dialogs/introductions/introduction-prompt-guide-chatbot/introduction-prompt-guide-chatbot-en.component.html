This is a guide to how prompts for chatbots should be worded to get the best possible responses.

  1. Ask one clear question or make one request
  2. Use proper language without technical terms or complex words
  3. Make sure the prompt is related to the chatbot's purpose
  4. Provide enough information for the chatbot to understand
  5. Don't ask multiple things in one prompt
  6. Use proper grammar and spelling
  7. Avoid slang, abbreviations, or emojis

Good vs. Bad Prompts for Asking Questions to the chatbot

  1. What is the mathematical equation for calculating the maximum flow in a network using the Ford-Fulkerson algorithm?
    "How does the Ford-Fulkerson algorithm work?" (Too broad and lacks specific details)
  2. Can you provide a list of important machine learning algorithms used in computer vision and explain their use cases?
    "What are some cool computer vision things?" (Too vague and lacks academic relevance)
  3. What is the difference between the stochastic gradient descent and batch gradient descent optimization algorithms?
    "Which one is better, SGD or BGD?" (Too simplistic and lacks the need for explanation)
