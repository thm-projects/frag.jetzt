Guide to use the Chatbot

This guide provides tips for creating effective prompts for chatbot, which can help users get the best possible answers. The chatbot is trained with English data and for better results, it is good to write prompts in English

Clarity and simplicity:

Write prompts that are easy for the chatbot to understand. Use simple language, avoid complex words or technical terms, and be concise. Ask one clear question or request at a time.

  • Better ✅: "What is the temperature in Berlin?"
  • Less effective ❌: "What's the weather like in Berlin? Is it cold or hot?"

In this example, the good prompt is clear, to the point and asks for a specific piece of information. The bad prompt is less clear and includes unnecessary details.

Relevance:

Write prompts that are relevant to the purpose of the chatbot. This will ensure that the chatbot provides accurate and helpful answers.

  • Better ✅: "What is the price of bitcoin today?"
  • Less effective ❌: "What is the population of China?"

In this example, the good prompt is relevant to a chatbot providing information about cryptocurrencies. The bad prompt is not relevant to the purpose of the chatbot.

Context:

Provide enough information in your prompt for the chatbot to understand what you're asking. This can include relevant details such as names, dates or locations.

  • Better ✅: "Can you recommend a restaurant in Hamburg that serves Italian food?"
  • Less effective ❌: "What's good to eat in Hamburg?"

In this example, the good prompt provides context by specifying the type of restaurant and location. The bad prompt is too vague and doesn't provide enough information for the chatbot to understand.

Avoid ambiguity:

Ask one clear question or make one request at a time. Avoid asking multiple questions or making multiple requests in one prompt. This can cause confusion and lead to inaccurate or unhelpful responses.

  • Better ✅: "What is the capital of Germany?"
  • Less effective ❌: "What is the capital of Germany and what is the population of Germany?"

In this example, the good prompt asks one clear question, whereas the bad prompt contains multiple questions.

Grammar and spelling:

Use correct grammar and spelling when writing your prompts. This will help the chatbot understand what you are asking and provide accurate answers.

  • Better ✅: "How do I reset my password?"
  • Less effective ❌: "How I rest my pasword?"

In this example, the good prompt uses correct grammar and spelling, while the bad prompt contains grammatical errors.

Avoid slang and abbreviations:

Avoid using slang, abbreviations, or emoticons in your prompts. These can be difficult for the chatbot to understand and can lead to inaccurate or unhelpful responses.

  • Better ✅: "What does 'zeitgeist' mean?"
  • Less effective ❌: "Wut does zeitgeist mean?"

In this example, the good prompt uses correct language, while the bad prompt uses slang and abbreviations.

Code Generation Specific-Use:

  1. Use clear and concise language without technical jargon or complex terminology
  2. Make sure the prompt is related to the desired code pattern or language
  3. Provide specific hints or "leading words" to guide the model towards the desired code pattern or language
  4. Use proper syntax and formatting for the desired code language
  5. Avoid using multiple prompts in one code example
  • Better ✅: "Write a Python function that takes in a list of integers and returns the sum of all even numbers."
  • Less effective ❌: "Create a function that adds up even integers in a list of numbers"

In this example, the good prompt is clear and specific, and provides a hint to the model to start coding in Python, while the bad prompt lacks specific details and doesn't provide any hints for the model to start coding in Python.

Other examples for Good vs. Bad Prompts for Asking Questions to the chatbot

  1. Better ✅: What is the mathematical equation for calculating the maximum flow in a network using the Ford-Fulkerson algorithm?
    2. Less effective ❌: "How does the Ford-Fulkerson algorithm work?" (Too broad and lacks specific details)
  2. Better ✅: Can you provide a list of important machine learning algorithms used in computer vision and explain their use cases?
    3. Less effective ❌: ""What are some cool computer vision things?" (Too vague and lacks academic relevance)
  3. Better ✅: What is the difference between the stochastic gradient descent and batch gradient descent optimization algorithms?
    4. Less effective ❌: "Which one is better, SGD or BGD?" (Too simplistic and lacks the need for explanation)
