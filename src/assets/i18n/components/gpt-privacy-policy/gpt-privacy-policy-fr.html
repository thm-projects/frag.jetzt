Ci-dessous, nous vous informons sur le traitement de vos données personnelles lors de l'utilisation de l'OpenAI API.

Finalités du traitement et base juridique

L'OpenAI API ne traite aucune donnée personnelle vous concernant.

Transfert de données

Vos données sont traitées sur des serveurs situés en dehors de l'Union européenne. Vos données personnelles ne seront pas divulguées à des tiers, sauf si la loi l'exige.

Stockage des données

Les données transmises via l'OpenAI API seront stockées à des fins de surveillance des abus et des utilisations abusives pendant une période maximale de 30 jours, sauf si une période de conservation plus longue est exigée par la loi.

Vos droits

Vous avez le droit d'information, de rectification, d'effacement, de limitation du traitement et de portabilité des données. Vous avez également le droit de retirer votre consentement à tout moment, sans affecter la légalité du traitement basé sur le consentement avant son retrait. Vous avez également le droit de déposer une plainte auprès d'une autorité de contrôle.

Si vous avez des questions concernant le traitement de vos données personnelles ou si vous souhaitez exercer l'un de vos droits concernant vos données personnelles, veuillez nous contacter à l'adresse openai.com/policies/privacy-policy.
