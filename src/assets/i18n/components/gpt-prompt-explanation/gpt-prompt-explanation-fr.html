1. Qu'est-ce qu'un Prompt ?

    Un Prompt est un texte donné en entrée à un modèle d'IA pour générer une réponse spécifique. En ce qui concerne ChatGPT, un Prompt est saisi par l'utilisateur et utilisé par le modèle pour générer une réponse. Le Prompt peut être une question, une phrase ou quelques mots qui aident le modèle à comprendre le contexte et le résultat souhaité.

  2. Qu'est-ce que les Prompt-Presets ?

    Les Prompt-Presets sont des entrées de texte prédéfinies qui peuvent être présentées à un modèle ChatGPT pour générer des types de réponses spécifiques. Ces préconfigurations peuvent aider ChatGPT à réagir de manière particulière et à fournir des réponses spécifiques. Les Prompt-Presets peuvent être utilisés, par exemple, pour générer des questions sur un sujet spécifique ou pour utiliser un style ou une langue particulière. L'utilisation de Prompt-Presets permet de contrôler et d'influencer les résultats de ChatGPT de manière ciblée.

  3. Que signifient les Prompt-Presets Contexte, Thème et Longueur sur la plateforme Q&R frag.jetzt ?

    • Contexte :

      Il s'agit des informations que l'utilisateur devrait inclure dans la question pour aider le chatbot à mieux comprendre le contexte de la question et ainsi générer une réponse plus précise. Un exemple de contexte pourrait être : j'ai essayé de réserver un vol mais le site web me montre toujours un message d'erreur.

    • Thème :

      Le thème décrit le sujet général ou la catégorie de la question ou du problème. En relation avec le contexte mentionné ci-dessus, le thème pour cette question pourrait être "voyages et réservation de vols".

    • Longueur :

      La longueur indique si la réponse de ChatGPT doit être courte, détaillée ou exhaustive.

      Dans une réponse courte, ChatGPT essaiera de répondre à la question avec le moins de mots possible. La réponse courte peut ne pas répondre correctement à la question.

      Dans une réponse détaillée, ChatGPT essaiera de répondre à la question aussi en détail que possible.

      Dans une réponse exhaustive, ChatGPT essaiera de générer une réponse avec toutes les informations connues en rapport avec la question. Il est possible que ChatGPT affiche également des informations non pertinentes.
